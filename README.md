# Festival DevOps Music Fest 2026

## Descripcion
Proyecto web del Festival DevOps Music Fest, desarrollado con frontend HTML/CSS y backend Flask, gestionado con Docker y Git.

## Tecnologias usadas
- HTML y CSS (frontend)
- Python + Flask (backend)
- Docker y Docker Compose (contenedores)
- Git y GitHub (control de versiones)

## Estructura del proyecto
- frontend/ ? pagina web del festival
- backend/ ? API Flask
- docker-compose.yml ? configuracion de contenedores
- README.md ? documentacion del proyecto

## Ramas creadas
- main ? rama principal
- feature-landing ? landing page
- feature-backend ? API Flask

## Como ejecutar
1. Clonar el repositorio
2. Ejecutar docker-compose up
3. Abrir http://localhost en el navegador
